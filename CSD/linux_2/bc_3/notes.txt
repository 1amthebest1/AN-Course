tr [a-z] [A-Z] uses the Unix tr (translate) command to convert lowercase letters to uppercase.

Example:

echo "hello world" | tr [a-z] [A-Z]

Output:

HELLO WORLD
How it works
tr reads characters from standard input.
[a-z] is the source character set (lowercase letters).
[A-Z] is the destination character set (uppercase letters).
Each character is translated positionally:

--

tr [a-z] [A-Z] << yo
> asa
> yo
ASA
tr [a-z] [A-Z] <<< "bhenchod gym jao"
BHENCHOD GYM JAO
