172.17.0.0/20 dev eth0 proto kernel scope link src 172.17.1.239

/20 means first 20 bit are reserved for network, and the remaining 12 bits is which can be given to hosts.

I think this means all the things on box, having that ip range should use my IP [given to me by router], as their default gateway.
